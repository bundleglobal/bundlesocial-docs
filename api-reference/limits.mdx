## Implementation Notes

### File Type Definitions

- **video** - Only videos allowed
- **image** - Only images allowed
- **image-or-video** - Images OR videos, but not both in the same post
- **image-and-video** - Images and/or videos can be mixed
- **image-or-video-or-document** - Images, videos, or documents (only one type per post)

### Aspect Ratio Calculation

```typescript
aspectRatio = width / height
```

### Bitrate Calculation

```typescript
// Bitrate in bits per second
bitrate = (fileSize * 8) / videoLength
```
