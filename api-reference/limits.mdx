description: Upload and post data limits
---

## Upload limits
### Images
- Maximum size: 25 MB
- Mime types: image/jpg, image/jpeg, image/png

### Videos
- Maximum size: 1 GB
- Mime types: video/mp4

## Posting limits
- Twitter
  - text - 280 characters (25,000 for X Premium subscribers)
- Pinterest
  - text - 100 characters
  - description - 800 characters
  - link - 2,048 characters
  - altText - 500 characters
  - note - 500 characters
- Facebook
  - text - 50,000 characters
- Instagram
  - text - 2,000 characters
- Threads
  - text - 500 characters
- TikTok
  - text - 150 characters
- LinkedIn
  - text - 1,300 characters
- YouTube
  - text - 100 characters
  - description - 5,000 characters
- Reddit
  - text - 300 characters
  - description - 30,000 characters
  - link - 2,048 characters
- Discord
  - text - 2,000 characters
  - username - 80 characters
  - avatarUrl - 2,048 characters
- Slack
  - text - 30,000 characters
  - username - 80 characters
  - avatarUrl - 2,048 characters
- Mastodon
  - text - 30,000 characters
  - spoiler - 50 characters
- Bluesky
  - text - 300 characters
  - tags - up to 8 tags, max 64 characters per tag
  - externalUrl - 2,048 characters
  - externalTitle - 300 characters
  - externalDescription - 1,000 characters
  - videoAlt - 10,000 characters
- Google Business Profile
  - text - 1,500 characters
  - eventTitle - 58 characters
  - offerCouponCode - 58 characters
  - offerTermsConditions - 1,500 characters
