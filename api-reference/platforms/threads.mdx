Threads is the new kid on the block, integrated tightly with Instagram's infrastructure.

## Quirks & Gotchas

### Terminology
*   **Comments** are called **Replies**.
*   **Shares** include both **Reposts** and **Quotes**.

## Analytics

### Profile Analytics
**Period:** Rolling window.

| Metric | Description | Note |
| :--- | :--- | :--- |
| `Impressions` | Views | Includes Quote count |
| `Impressions (Unique)` | Unique views | Includes Repost count |
| `Views` | Views | |
| `Likes` | Likes | |
| `Comments` | Replies | |
| `Followers` | Followers | |
| `Following` | Returns `0` | Not provided |

**Period:** Lifetime.
