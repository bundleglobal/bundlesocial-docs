Mastodon is part of the Fediverse. It's decentralized, which means "analytics" is a loose concept.

## Quirks & Gotchas

### Terminology
*   **Likes** = **Favourites**.
*   **Shares** = **Reblogs**.

---

## Analytics

### Profile Analytics
**Period:** Snapshot.

| Metric | Description | Note |
| :--- | :--- | :--- |
| `Post Count` | Total toots | |
| `Followers` | Followers | |
| `Following` | Following | |
| `Impressions` | Returns `0` | Not available |
| `Likes` | Returns `0` | Not available |

### Post Analytics
**Period:** Lifetime.

| Metric | Description | Note |
| :--- | :--- | :--- |
| `Likes` | Favourites | |
| `Comments` | Replies | |
| `Shares` | Reblogs | |
